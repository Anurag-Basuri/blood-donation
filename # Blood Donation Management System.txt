# Blood Donation Management System

A comprehensive platform connecting blood donors, recipients, NGOs, hospitals, and blood banks to streamline the blood donation process and save lives.

---

## Why This Project Matters

- **Increase Blood Donation Rates:** Makes it easy to find donation centers and schedule appointments, encouraging more regular donors.
- **Improve Blood Supply Chain Management:** Optimizes blood distribution to ensure availability where and when needed.
- **Reduce Blood Wastage:** Minimizes expired blood through efficient inventory management.
- **Enhance Transparency and Trust:** Secure, transparent platform for donors and recipients, fostering trust in the process.
- **Empower Patients and Providers:** Real-time blood availability info for informed treatment decisions.

---

## Features

- User, NGO, and Hospital registration & authentication
- Role-based dashboards
- Blood donation request & management
- Blood camp and blood bank management
- Real-time donor-recipient matching
- Certificates, badges, and donation history
- Responsive, modern UI with statistics and testimonials

---

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Chart.js, Google Maps API
- **Backend:** Node.js, Express, MongoDB (Mongoose), JWT, Cloudinary, Nodemailer

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- MongoDB instance

### Backend

1. Install dependencies:
   ```
   cd Backend
   npm install
   ```
2. Create a `.env` file (see sample in `Backend/`).
3. Start the backend:
   ```
   npm run dev
   ```

### Frontend

1. Install dependencies:
   ```
   cd Frontend
   npm install
   ```
2. Create a `.env` file (see sample in `Frontend/`).
3. Start the frontend:
   ```
   npm run dev
   ```

## API Overview

- **User:** Registration, login, profile, donation history, statistics
- **NGO:** Registration, login, camp management, inventory, requests
- **Hospital:** Registration, login, blood requests, profile
- **Admin:** Dashboard, user/NGO/hospital management, analytics
- **Blood:** Donation, inventory, status updates
- **Centers:** Blood banks and donation camps
